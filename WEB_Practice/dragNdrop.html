reduce() arr.reduce(callback함수, 초기값)
*callback 함수의 매개변수
   accumulatr(누산기):이전 반환값. 초기값을 제공했을경우 그 초기값
   currentValule(현재 값)
   currentIndex(처리할 현재 값 인덱스) : 디폴트1부터(초기값이 제공되었다면 0부터인거)
*초기값
   callback함수 최초 호출에 제공하는 값. 제공 안했다면 배열 첫번째 요소
[EX]
[0, 1, 2, 3, 4].reduce( (prev, curr) => prev + curr );
>>> 10
[0, 1, 2, 3, 4].reduce( (prev, curr) => prev + curr ,10 );
>>> 20
