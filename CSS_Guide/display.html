DISPLAY

Display is key to layout of a page
• The display property is the most important CSS property for controlling layout
• The display property specifies if/how an element is displayed
• Display affects the layout of neighboring elements

Block & Inline

div 한줄을 차지합니다. 가로세로 지정가능

span1 줄바꿈이 되지 않습니다. 가로세로 지정불가 span2 span3
어느 태그에서든 css에서 display속성을 이용할 수 있다.

display:block (div,p,h 등) 가로세로 지정가능

display:inline (span, a 등) 가로세로 지정불가

display:inline-block 줄바꿈이되지않고 가로세로 지정가능
